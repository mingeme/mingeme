# 🚀 Mingeme

> 让创建表情包变得简单有趣 | Make meme creation simple and fun

## 📖 项目介绍

Mingeme 是一个简单易用的表情包制作工具，让你轻松创建、编辑和分享有趣的表情包。

## ✨ 特性

- 🎨 简洁直观的用户界面
- 🖼️ 支持多种图片格式
- ✍️ 自定义文字和样式
- 💾 一键保存分享
- 📱 支持移动端适配

## 🛠️ 技术栈

- Frontend: React.js
- Backend: Python
- Database: SQLite
- UI Framework: Tailwind CSS

## 🚀 快速开始

### 环境要求

- Node.js >= 16
- Python >= 3.8

### 安装步骤

1. 克隆仓库
```bash
git clone https://github.com/yourusername/mingeme.git
cd mingeme
```

2. 安装依赖
```bash
# 前端依赖
npm install

# 后端依赖
pip install -r requirements.txt
```

3. 运行项目
```bash
# 启动前端
npm run dev

# 启动后端
python app.py
```

## 📝 使用说明

1. 上传图片或选择模板
2. 添加文字和表情
3. 调整样式和位置
4. 一键保存或分享
